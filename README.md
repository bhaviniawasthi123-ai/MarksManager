
# Student Marks Management System (Java)

A simple console-based Java application that allows users to enter student names and marks, view all stored records, and save them into a text file for future reference.  
This project was developed as part of the *Java - Build Your Own Project* requirement.



## Features

###  Add Student  
Enter student name and marks (0–100). Validates input.

###  View Students  
Displays all stored student records in a clean format.

###  Save Records to File  
Automatically generates students.txt containing all student details.

###  Menu-Driven  
Simple and user-friendly console interface.



##  Technologies Used

- Java (Core Java)
- Eclipse IDE
- File Handling (FileWriter, BufferedWriter)
- ArrayList for in-memory storage

##  How to Run the Project

1. Install *JDK 8 or above*
2. Open the project in *Eclipse* or any Java IDE
3. Run the main file:

MarksOfStudent.java

4. Follow the on-screen menu:
- Add Student  
- View Students  
- Save to File  
- Exit
